Hướng Dẫn Sử Dụng

1. Quản Lý Thay Thế

Thêm các cặp từ cần thay đổi. Hỗ trợ Regex (Biểu thức chính quy). Ví dụ: Tìm \s+ thay bằng dấu cách để xóa khoảng trắng thừa.

2. Format Hội Thoại

Hệ thống tự động nhận diện cấu trúc Nhân vật: "Lời nói" để định dạng lại theo chuẩn bạn chọn.

3. Chia Chương

Dán file truyện full, chọn chia theo số lượng chương cố định hoặc dùng Regex Chương \d+ để tách tự động.
